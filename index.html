What I've Been Doing in EDD

by Nick Alvarez

Hi, I'm Nick. I'm a high school senior in Mr. Eno's Engineering Design and Development class.

I am interested in web development, so I spent the past few months in EDD learning and practicing HTML, CSS, and JavaScript. I also experimented with AI and took a dive in web design.

I'm someone who pivots from project to project a lot, so I've done multiple "mini projects" this semester. Here's a web portfolio of all the projects and prototypes I've worked on.

Thanks for stopping by my exhibit.

Web Builder Bot
Prompt Engineering Project

I prompt engineered a chat-bot to provide blocks of HTML, CSS, and JavaScript code for users who need help in web-development.

To encourage users to explore programming on their own, the chat bot provides detailed explanations of how each element of the generated code works.

You can access the bot here: https://www.playlab.ai/project/clq5jf2ai05kpq8969pu5psls

Digital Detox
AI-Assisted Web Design and Development Project

A Front-End Development project where I created a proof-of-concept page for a web tool that gives users offline tasks to do as a way to balance out how much we use screens and technology in day-to-day life.

My Web Builder Bot generated most of the webpage's framework. I then modified the code to make the site look the way I wanted it to. I used Canva's AI-image generator (the only AI image generation tool that I could access on school wifi) to make pictures for the site.

I uploaded my prototype as a Github Repository. You can access it here: https://liontrovert.github.io/ProjectLeoDigitalDetoxv1/

Web Development & Javascript Courses
Self-paced learning with FreeCodeCamp and Codecademy

This year, I learned the ropes in HTML and CSS by taking courses in FreeCodeCamp's Responsive Web Design program, introduced to me by my Computer Science Seminar teacher, Justin Mitchell.

At the beginning of second semester, I switched to Codeacdemy (which I believe is better) to learn JavaScript. My CS teacher was able to give us free access to Codecademy's premium features, so I used it to access the "Create a Front-End App with React" skill path, which I'm chipping away at. The best thing about Codecademy is that it offers IDE's to program in - called Workspaces. In fact, I used Workspaces to make this site.

Wix Creators of Tomorrow
Current Project: A Website to submit to the Wix Creators of Tommorrow Competition

After learning Wix software, I started working on a website that showcases skills and careers in digital media for Wix's "Creators of Tomorrow" competition, due April 13. This website combines my current career interest in media/writing and my interest in web development.
